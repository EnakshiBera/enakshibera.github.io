# enakshibera.github.io

This repository hosts my personal academic website using GitHub Pages.

The purpose of this repository is **infrastructure**, not research content or code.
It exists to serve static web pages (HTML/CSS) that present my academic profile
(CV, research, and teaching) at:

https://enakshibera.github.io

---

## What this repository is

- A **static website** built with plain HTML and CSS  
- Hosted via **GitHub Pages**
- Used to share:
  - Academic CV (PDF)
  - Research overview
  - Teaching experience

There is **no backend**, no JavaScript framework, and no build system.

---

## What this repository is NOT

- Not a research code repository
- Not a data repository
- Not a software project
- Not meant for external contributors

All research code and data live elsewhere.

---

## File structure

- `index.html`  
  Homepage / landing page

- `research.html`  
  Research overview page

- `teaching.html`  
  Teaching experience page

- `cv.html`  
  CV page (links to PDF)

- `style.css`

- `Enakshi_Bera_CV.pdf`

## Design philosophy

- Minimal and academic
- Content-first, not design-heavy
- Pages are updated incrementally
- Sections may be temporarily hidden or simplified while under revision

A placeholder page may be used when content is not ready for public view.
